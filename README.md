# -InkTrace-Multimodal-KG-Display-Platform-for-Calligraphers-
本作品面向书法初学者与爱好者，针对书法学习中“资料分散、功能单一、图文割裂”的痛点，开发了“墨谱”，将书法知识转化为可检索、可交互的智能服务。  平台集成三大核心功能：知识图谱可视化检索，展示书法家的生平、师承与交游网络；单字多书家风格比对，同屏对比不同书家的单字书写差异；AI字体流派识别，基于深度学习自动分析上传图片的风格流派，准确率达89.74%。  技术层面，后端采用Flask框架，知识图谱基于Neo4j，前端使用ECharts，AI模块基于ResNet50迁移学习。平台已完成三本古籍的知识抽取，形成含11类关系、10类实体的标准化知识图谱。  本作品首次将三大功能融合于一体，已部署为可访问的Web平台，可应用于个人学习、院校教学与文化传播，为“AI+传统文化”提供了可参考的实践范式。
